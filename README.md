# Qribbble
This is a third-party client for Dribbble website
此应用为dribbble网站的第三方客户端
为本人练手学习作品，目前具备点赞， 登录， 浏览作品等功能 还有诸多不完善之处；
已更新 优化登陆 新增设置界面 新增推送功能（beta）新增探索界面 新增人物详情界面 以及follow或者unfollow功能。
