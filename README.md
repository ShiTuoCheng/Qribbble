# Qribbble
This is a third-party client for Dribbble website<br>
此应用为dribbble网站的第三方客户端<br>
为本人练手学习作品，目前具备点赞， 登录， 浏览作品等功能 还有诸多不完善之处；<br>
-------
已更新:<br> 
优化登陆<br>
新增设置界面<br>
新增推送功能（beta）<br>
新增探索界面<br>
新增人物详情界面<br>
以及follow或者unfollow功能。<br>
